- Sequence Types

#### Naming Rule
- 영문 알파벳, 언더스코어(_), 숫자로 구성
- 숫자로 시작할 수 없음
- 대소문자를 구분
- 특정 키워드는 파이썬 내부 예약어로 사용할 수 없음
    - ex) for, False, and, assert, yield...

#### Variable, Value, Memory
- 변수는 그 변수가 참조하는 객체의 메모리 주소를 가짐

- 객체 (Object)
    - 타입을 갖는 메모리 주소 내 값
    - "값이 들어있는 상자"
